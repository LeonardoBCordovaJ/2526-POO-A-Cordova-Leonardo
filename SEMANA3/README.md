# Comparación: Programación Tradicional vs POO - Promedio Semanal del Clima

## Descripción del Proyecto

Este proyecto implementa un sistema para calcular el promedio semanal de temperaturas utilizando dos enfoques de programación:

1. **Programación Tradicional** (basada en funciones)
2. **Programación Orientada a Objetos - POO** (basada en clases y objetos)

## Objetivo

Demostrar las diferencias entre ambos paradigmas de programación y comprender las ventajas de la POO en términos de organización, reutilización y mantenimiento del código.
